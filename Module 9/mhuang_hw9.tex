We start with 
\[
  \int_{-\infty}^{\infty} \exp\{-x^2\} \exp\{\frac{-(x - \mu)^2 y}{2\sigma^2}\} dx
\]
\[
  = \int_{-\infty}^{\infty} \exp\{-x^2 - \frac{(x - \mu)^2 y}{2\sigma^2}\} dx
\]
\[
  = \int_{-\infty}^{\infty} \exp\{-x^2 - \frac{(x^2 + \mu^2 - 2x\mu) y}{2\sigma^2}\} dx
\]
\[
  = \int_{-\infty}^{\infty} \exp\{-x^2 - \frac{x^2y}{2\sigma^2} - \frac{y\mu^2}{2\sigma^2} + \frac{xy\mu}{\sigma^2}\} dx
\]
\[
  = \int_{-\infty}^{\infty} \exp\{-(1 + \frac{y}{2\sigma^2})x^2 + \frac{xy\mu}{\sigma^2} - \frac{y\mu^2}{2\sigma^2} \} dx
\]
With this expression equating to the form of \(-ax^2 + bx + c\), we can try to complete the square as instructed, which would yield \(-a(x - \frac{b}{2a})^2 + \frac{b^2}{4a} + c\). Let's try to match the format with \(a = 1 + \frac{y}{2\sigma^2}\), \(b = \frac{y\mu}{\sigma^2}\), and \(c = -\frac{y\mu^2}{2\sigma^2}\):
\[
  \frac{b}{2a} = \frac{\frac{y\mu}{\sigma^2}}{2 + \frac{y}{\sigma^2}}
\]
\[
  = \frac{\frac{y\mu}{\sigma^2}}{\frac{2\sigma^2 + y}{\sigma^2}}
\]
\[
  = \frac{y\mu}{2\sigma^2 + y}
\]
and
\[
  \frac{b^2}{4a} = \frac{\frac{y^2\mu^2}{\sigma^4}}{4 + \frac{2y}{\sigma^2}}
\]
\[
  = \frac{\frac{y^2\mu^2}{\sigma^4}}{\frac{4\sigma^2 + 2y}{\sigma^2}}
\]
\[
  = \frac{y^2\mu^2}{4\sigma^4 + 2y\sigma^2}
\]
So plugging in:
\[
  \int_{-\infty}^{\infty} \exp\{ -(1 + \frac{y}{2\sigma^2})(x - \frac{y\mu}{2\sigma^2 + y})^2 + \frac{y^2\mu^2}{4\sigma^4 + 2y\sigma^2} - \frac{y\mu^2}{2\sigma^2} \} dx
\]
\[
  = \int_{-\infty}^{\infty} \exp\{ -(1 + \frac{y}{2\sigma^2})(x - \frac{y\mu}{2\sigma^2 + y})^2 + \frac{y^2\mu^2 - y\mu^2(2\sigma^2 + y)}{2\sigma^2(2\sigma^2 + y)} \} dx
\]
\[
  = \int_{-\infty}^{\infty} \exp\{ -(1 + \frac{y}{2\sigma^2})(x - \frac{y\mu}{2\sigma^2 + y})^2 + \frac{y^2\mu^2 - 2y\mu^2\sigma^2 - y^2\mu^2}{2\sigma^2(2\sigma^2 + y)} \} dx
\]
\[
  = \int_{-\infty}^{\infty} \exp\{ -(1 + \frac{y}{2\sigma^2})(x - \frac{y\mu}{2\sigma^2 + y})^2 + \frac{- 2y\mu^2\sigma^2}{2\sigma^2(2\sigma^2 + y)} \} dx
\]
\[
  = \int_{-\infty}^{\infty} \exp\{ -(1 + \frac{y}{2\sigma^2})(x - \frac{y\mu}{2\sigma^2 + y})^2 + \frac{-y\mu^2}{2\sigma^2 + y} \} dx
\]
\[
  = \exp\{\frac{-y\mu^2}{2\sigma^2 + y}\} \int_{-\infty}^{\infty} \exp\{ -(1 + \frac{y}{2\sigma^2})(x - \frac{y\mu}{2\sigma^2 + y})^2 \} dx
\]
The integral is the Gaussian integral, with the form \(\int_{-\infty}^{\infty} e^{-ax^2} dx = \sqrt{\frac{\pi}{a}}\). Using this:
\[
  = \exp\{\frac{-y\mu^2}{2\sigma^2 + y}\} \sqrt{\frac{\pi}{1 + \frac{y}{2\sigma^2}}}
\]
\[
  = \exp\{\frac{-y\mu^2}{2\sigma^2 + y}\} \sqrt{\frac{\pi}{\frac{2\sigma^2 + y}{2\sigma^2}}}
\]
\[
  = \exp\{\frac{-y\mu^2}{2\sigma^2 + y}\} \sqrt{\frac{2\pi\sigma^2}{2\sigma^2 + y}}
\]
With the additional term in the corrected version of the problem:
\[
  = \frac{1}{\sqrt{\frac{2\pi\sigma^2}{y}}}\exp\{\frac{-y\mu^2}{2\sigma^2 + y}\} \sqrt{\frac{2\pi\sigma^2}{2\sigma^2 + y}}
\]
\[
  = \sqrt{\frac{y}{2\pi\sigma^2}}\exp\{\frac{-y\mu^2}{2\sigma^2 + y}\} \sqrt{\frac{2\pi\sigma^2}{2\sigma^2 + y}}
\]
\[
  = \sqrt{\frac{y}{2\pi\sigma^2} \cdot \frac{2\pi\sigma^2}{2\sigma^2 + y}}\exp\{\frac{-y\mu^2}{2\sigma^2 + y}\}
\]
\[
  = \sqrt{\frac{y}{2\sigma^2 + y}}\exp\{\frac{-y\mu^2}{2\sigma^2 + y}\}
\]
\[
  = \sqrt{\frac{y}{2\sigma^2 + y}}\exp\{\frac{-y\mu^2}{2\sigma^2 + y}\}
\]
\[
  = \frac{1}{\sqrt{\frac{2\sigma^2 + y}{y}}}\exp\{\frac{-y\mu^2}{2\sigma^2 + y}\}
\]
\[
  = \frac{1}{\sqrt{\frac{2\sigma^2}{y} + 1}}\exp\{\frac{-y\mu^2}{2\sigma^2 + y}\}
\]
\[
  = \frac{1}{\sqrt{\frac{2\sigma^2}{y} + 1}}\exp\{\frac{-y\mu^2}{2\sigma^2 + y}\}
\]
\[
  = \frac{1}{\sqrt{\frac{2\sigma^2}{y} + 1}}\exp\{-\frac{\mu^2}{\frac{2\sigma^2 + y}{y}}\}
\]
\[
  = \boxed{\frac{1}{\sqrt{\frac{2\sigma^2}{y} + 1}}\exp\{-\frac{\mu^2}{1 + \frac{2\sigma^2}{y}}\}}
\]
exactly as we sought to show in the corrected version of the problem.

\section*{5.}

Assuming that we use antithetic variables to estimate some parameter of the standard normal distribution, we must prove that the covariance between \(X_i\) and \(Y_i\) is -1, i.e. \(\text{Cov}(X_i, Y_i) = -1\). \\ \\
We set up with \(U \sim \text{Unif}(0, 1)\). Using the hint with the normal distribution being symmetric about 0, we know that \(\Phi(z) = 1 - \Phi(-z)\) for the range of \(U\), and equivalently that 
\[
  x = \Phi(z) = 1 - \Phi(-z) 
\]
\[
  \Phi(-z) = 1 - x
\]
\[
  -z = \Phi^{-1}(1 - x)
\]
\[
  -\Phi^{-1}(-x) = \Phi^{-1}(1 - x)
\]
So taking advantage of this, let's set \(X = \Phi^{-1}(U)\) and \(Y = \Phi^{-1}(1 - U)\) accordingly. We can then determine that 
\[
  Y = \Phi^{-1}(1 - U) = -\Phi^{-1}(-U) = -X
\]
So therefore 
\[
  \text{Cov}(X_i, Y_i) = \text{Cov}(X_i, -X_i) = -\text{Cov}(X_i, X_i) 
\]
\[
  = -\text{Var}(X_i) = \boxed{-1}
\]
just as we aimed to prove.
